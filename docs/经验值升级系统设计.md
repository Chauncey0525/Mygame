# 经验值升级系统设计文档

## 一、系统概述

### 1.1 设计目标
- 让玩家有清晰的成长路径
- 关卡难度与玩家等级匹配
- 新玩家能顺利开始游戏
- 老玩家有持续追求

### 1.2 核心机制
- **玩家等级**：影响关卡解锁、角色上限
- **经验获取**：战斗、任务、道具
- **等级解锁**：关卡、功能逐步开放

## 二、玩家等级系统

### 2.1 等级上限

- 后续版本可扩展

### 2.3 经验计算公式

```python
def get_exp_to_next_level(level):
    """计算升到下一级所需经验"""
    if level < 5:
        # 新手期：快速增长
        return 100 + (level - 1) * 50
    elif level < 10:
        # 过渡期
        return 300 + (level - 5) * 100
    elif level < 20:
        # 中期
        return 800 + (level - 10) * 200
    elif level < 30:
        # 后期
        return 3000 + (level - 20) * 300
    elif level < 40:
        # 高级
        return 6000 + (level - 30) * 500
    else:
        # 顶级
        return 12000 + (level - 40) * 1000
```

## 三、关卡解锁系统

### 3.1 章节解锁条件

| 章节 | 解锁条件 | 关卡难度 |
|------|----------|----------|
| 第1章 | 默认解锁 | 推荐等级 1-5 |
| 第2章 | 玩家等级 ≥ 10 | 推荐等级 10-15 |
| 第3章 | 玩家等级 ≥ 20 + 通关第2章 | 推荐等级 20-25 |
| 第4章 | 玩家等级 ≥ 30 + 通关第3章 | 推荐等级 26-30 |

### 3.2 关卡进入条件

- 玩家等级 ≥ 关卡推荐等级 - 3（允许适当越级）
- 体力满足消耗要求
- 前置关卡已通关

### 3.3 关卡经验奖励

| 关卡难度 | 基础经验 | 首通奖励 |
|----------|----------|----------|
| 简单 | 50-100 | +50%经验 |
| 普通 | 100-200 | +50%经验 |
| 困难 | 200-400 | +100%经验 |
| 地狱 | 400-800 | +100%经验 |

## 四、新玩家引导

### 4.1 初始资源

新玩家注册后获得：
- **金币**: 10,000
- **钻石**: 1,000
- **召唤券**: 10张
- **体力**: 100（满）

### 4.2 初始角色

新玩家注册后自动获得**初始英雄选择**：
- 选择1个初始角色（从3个普通角色中选择）
- 或直接赠送一个随机普通角色

**初始角色池**（普通稀有度）：
- 步兵（战士）
- 弓箭手（远程）
- 见习法师（法师）

### 4.3 新手任务

完成新手引导任务获得额外奖励：
1. **完成首次召唤** → 获得1个史诗角色
2. **通关第1章** → 获得100钻石
3. **角色升到5级** → 获得5000金币

## 五、经验获取途径

### 5.1 战斗经验

| 来源 | 经验值 | 说明 |
|------|--------|------|
| 普通关卡 | 50-200 | 根据关卡难度 |
| 精英关卡 | 200-400 | 每日限次 |
| Boss关卡 | 400-800 | 每周刷新 |

### 5.2 任务经验

| 任务类型 | 经验值 | 刷新周期 |
|----------|--------|----------|
| 每日任务 | 50-100/个 | 每日 |
| 成就任务 | 200-1000 | 一次性 |
| 七日目标 | 100-500/个 | 新玩家7天 |

### 5.3 道具经验

| 道具 | 经验值 | 获取途径 |
|------|--------|----------|
| 智慧之书(小) | 100 | 关卡掉落 |
| 贤者之书(中) | 300 | 任务奖励 |
| 觉醒之书(大) | 1000 | 商店/活动 |
