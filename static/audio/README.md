# 音频资源

## 目录说明

- **bgm/**：背景音乐。放入 `main.ogg` 或 `main.mp3` 后，在游戏内会循环播放（需在导航栏点击喇叭取消静音后生效）。
- **sfx/**：音效文件（可选）。当前音效为脚本内置的占位音效（Web Audio 生成），若在此目录放置同名文件可覆盖（需在 `audio.js` 中扩展加载逻辑）。

## 占位音效（无需文件）

未放置任何文件时，以下音效由浏览器自动生成：

- 按钮点击、召唤（普通/稀有/史诗/传说）、战斗受击、胜利/失败等。

替换为自有素材时，请保持格式为 **ogg** 或 **mp3**，并控制文件大小以利加载。
